This Project is a Devise Google Sign-in integration.

# Prerequisites
1. Ruby 3.3.0
2. Rails 7.1.3
3. PostgreSQL

clone this repo and
Navigate to the project directory.

run these commands in terminal

1. `bundle install` 
2. `rails db:create`
3. `rails db:migrate`

Run the Project using

`./bin/dev`

Navigate to browser

`https://localhost:3000`
