This Project is a Devise Google Sign-in integration.

# Prerequisites
1. Ruby 3.3.0
2. Rails 7.1.3
3. PostgreSQL

clone this repo and
Navigate to the project directory.

run these commands step by step in terminal

``` 
bundle install
``` 
```
rails db:create
```
```
rails db:migrate
```

Run the Project using

```
./bin/dev
```

Navigate to browser

`https://localhost:3000`
